Model Checkpoints
=================

depth_anything_v2_vits.pth
    Depth Anything V2 VITS checkpoint used for monocular depth estimation.

maskrcnn_fpn_known_objects_super_final.pth
    Mask R-CNN with FPN checkpoint used for known-object instance
    segmentation.

zero3dce_best.pth
    Zero-3DCE checkpoint used for low-light frame enhancement.

These binary files are required to rerun model inference but are not required
to view the precomputed Hugging Face Space. Keep the filenames unchanged
unless the corresponding notebook or application configuration is updated.
